## 1. 概述

ResumePilot 是一个接入 ChatGPT 进行简历筛选的项目，旨在帮助 HR 和招聘经理提高简历筛选的准确性和效率。

## 2. 需求分析

ResumePilot 可以从指定存放简历的目标文件夹中解析简历并根据使用者输入的关键字进行筛选，输出符合要求的候选人。

## 3. 架构设计

在Github上找到一个符合这个项目的开源项目[Gradio](https://github.com/gradio-app/gradio)，可以完全使用Python来构建一个Web App，无需其他的语言即可实现前后端一体部署。

### 3.1 前端架构设计

前端主要包括以下组件：

* 筛选条件按钮：筛选目标条件按钮和条件输入框（包含必填条件和选填条件），且需要为筛选条件输入提供更加便利的操作。
* 读取筛选条件的按钮:为了防止网页刷新的时候会自动恢复页面，显示条件的框会清空内容，这时需要从Redis中读取网页刷新前的条件。
* 添加第二组专门使用gpt解析加分析简历内容是否符合的筛选条件输入框，可以根据人话的需求，让gpt来筛选匹配的简历并返回结果。
* 解析简历按钮：启动简历解析功能。
* 筛选结果展示组件：展示按照要求解析完成的简历的结果。
* 分析报告导出组件：导出解析完成的简历的表格和符合筛选条件的数据表。

### 3.2 后端架构设计

后端主要包括以下组件：

* 结果返回组件：返回结果并展示到前端界面上。
* ChatGPT解析简历模块：使用 ChatGPT 解析简历内容
* 分析结果生成模块：分析完全部简历并生成分析报告的模块
* Redis 解析结果存储模块
* 文本转Excel文件生成模块
* 使用Redis获取前端输入的筛选条件
* 图片PDF OCR解析模块

### 3.3 整体架构设计

前端通过 API 调用后端服务。后端服务包括简历解析、关键字匹配、上下文交互和结果显示四个组件，最终将筛选结果返回给前端展示。

### 3.4 使用环境要求

* 操作系统：Windows 10 或 Linux 系统。
* 浏览器：Chrome、Firefox 等现代浏览器。
* Python 版本：Python 3.7 及以上版本。
* Redis 版本：

## 6. 交互说明

前端界面将保留 ChatGPT 的对话界面，包含上传、聊天输入框和结果展示三大组件。所有复杂的运算均在后台实现，前端无需展示运算过程，只需展示结果即可。

### 7. 代码风格
1. 注释全部采用英文，包括与ChatGPT交互的内容。
2. 代码使用面向对象的方式进行模块功能的封装。
3. 中国服务器代理配置说明文档。
4. 前端界面需要添加一个切换语言的功能，采用I18N的标准。
